🏠 DESENVOLVIMENTO LOCAL (5 minutos)
─────────────────────────────────────

1️⃣ Requisitos:
   • PHP 7.4+ (com MySQLi)
   • MySQL 5.7+
   • Navegador moderno

2️⃣ Iniciar servidor:
   cd api-viacep
   php -S localhost:8000

3️⃣ Criar banco de dados:
   • Abra phpMyAdmin → http://localhost/phpmyadmin
   • Novo banco: viacep_db
   • Executar banco SQL em backend/database.sql

4️⃣ Configurar PHP:
   • Edite: backend/config.php
   • Deixe valores padrão (localhost, root, viacep_db)

5️⃣ Acessar:
   • http://localhost:8000
   • Teste com CEP: 01310100


🌍 DEPLOY INFINITYFREE (10 minutos)
────────────────────────────────────

1️⃣ Criar conta: https://www.infinityfree.net/

2️⃣ Criar banco de dados:
   • Painel >> MySQL Databases
   • Anotar credenciais

3️⃣ Criar tabelas:
   • phpmyadmin >> Executar SQL do arquivo

4️⃣ Atualizar config.php:
   • Substituir credenciais do InfinityFree

5️⃣ Upload via FTP:
   • FileZilla (grátis)
   • Host: ftp.infinityfree.com
   • Destino: /htdocs

6️⃣ Acessar:
   • https://seu-nome.infinityfree.app
