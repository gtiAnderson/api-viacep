Como Funciona

  • 1. Digite o CEP: Insira um CEP válido (8 dígitos, ex: 01310100)
  • 2. Busca Automática: Os dados são buscados automaticamente na API ViaCEP
  • 3. Preenchimento: O logradouro, bairro, cidade e estado são preenchidos automaticamente
  • 4. Complete: Adicione o número e complemento (se houver)
  • 5. Enviar: Clique em "Enviar" para gravar os dados no banco de dados
